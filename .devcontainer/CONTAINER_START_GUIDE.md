# Dev Container 起動手順

## 1. コンテナの起動

1. VSCodeでこのプロジェクトを開きます
2. 左下のステータスバーに「Dev Container で開く」と表示されているのでクリックします
3. コンテナのビルドと起動が開始されます
4. 初回起動時は数分かかる場合があります

## 2. 開発環境のセットアップ

コンテナ起動後、ターミナルで以下のコマンドを実行します：

```bash
./.devcontainer/setup.sh
```

このスクリプトは以下の処理を行います：
- 必要なディレクトリを作成
- `/server` の内容を `/api` にコピー
- 依存関係のインストール
- 必要なスクリプトの実行権限付与

## 3. サーバーの起動

セットアップ完了後、以下のコマンドでサーバーを起動できます：

```bash
npm run dev
```

これでAPIサーバーが5000番ポートで起動します。

## 4. Expo開発サーバーの起動

別のターミナルで以下のコマンドを実行します：

```bash
npx expo start
```

これでExpo開発サーバーが3000番ポートで起動します。

## 5. ブラウザでの確認

- APIサーバー: http://localhost:5000
- Expo開発サーバー: http://localhost:3000

## 6. Vercelデプロイ用

Vercelにデプロイする際は、以下のスクリプトを実行して `/server` の内容を `/api` にコピーします：

```bash
./.devcontainer/copy-server.sh
```

その後、package.jsonのスクリプトをVercel用に修正してください。
